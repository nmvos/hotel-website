# Benodigdheden
* Een IDE naar keuze
* Python
* Django

# Stappen om het project op te zetten
* Clone de repository
* Installeer de vereisten uit ```requirements.txt```
* Ga naar de juiste submap van het project met ```cd hotel_website```
* Start de applicatie met ```python manage.py runserver```
